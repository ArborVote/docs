Quadratic Voting

After the debating stage has ended, the voting phase begins. Each joined participant has the same number of initial vote tokens Tvote,0T_{\text{vote},0}Tvote,0​ which she/he can use to support or oppose arguments.

Depending on the voters preference strength PPP, he can put different weight WWW to her/his vote by spending more vote tokens

W(P)=C(P),W(P)= C(P),W(P)=C(P),

with CCC being the quadratic cost function The cost of the vote CCC scales quadratically with the preference strength

C(P)=P2⋅TvoteC(P) = P^2 \cdot T_\text{vote}C(P)=P2⋅Tvote​

For example, to double her/his vote weight from W=1W=1W=1 to W=2W=2W=2, the voter must spend C(2)=4 TvoteC(2)=4\,T_\text{vote}C(2)=4Tvote​ tokens.
